# Hubot Slack Training

## Environment

- Node.js ... 16.1.0


## How to use with slack in local

1. run below command

```sh
HUBOT_SLACK_TOKEN=$HUBOT_SLACK_TOKEN ./bin/hubot -a slack
```

2. please type in the hubot DM or channel inviting the hubot.
